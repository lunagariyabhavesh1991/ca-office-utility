Installation Guide

Follow these steps to set up the CA Office PDF Utility on your system.

Step 1: Download

Download the ZIP file for your operating system (Windows or macOS) from the main page.

Step 2: Extract

Extract the contents of the ZIP file to a dedicated folder, for example: C:\CA_Office_Utility

Step 3: Run

Double-click CA_Office_Utility.exe to launch the application. On first run, you will see an activation screen.

Step 4: Activation

Enter your license key or contact support to generate an Activation Key for your Machine ID.
